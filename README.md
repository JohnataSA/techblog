# techblog
TechBlog

Um projeto desenvolvido em HTML e CSS, inspirado em blogs de tecnologia.
O objetivo é praticar o desenvolvimento de páginas estáticas e aplicar conceitos básicos de estruturação e estilização de sites.
<img width="812" height="947" alt="image" src="https://github.com/user-attachments/assets/cadb1694-8c10-4b5e-888c-ee87a63e332d" />

#Funcionalidades:

-> Página principal com posts recentes.
-> Navegação entre categorias (Jogos, Celulares, Informática, Eletrônicos).
-> Área lateral com postagens recentes e lista de categorias.
-> Layout responsivo simples utilizando HTML e CSS puro.

#Tecnologias Utilizadas:

1. HTML5 – estrutura e semântica
2. CSS3 – estilização e layout
